# Web-Weather-Widget
Простой Web-виджет погоды для HTML/PHP-страницы

# **Инструкция по размещению и продвижению Web-страницы в поисковой системе Яндекс**

1. Размещение сайта на хостинге.
2. Размещение информации о сайте в сервисе Яндекс Вебмастер.
   2.1. Подтверждение прав на сайт.
   Подтверждение прав на Web-сайт, если он написан на чистом коде, лучше производить через метатег.
   Пример: <meta name="yandex-verification" content="fcf103b3d7284ca6" />, размещение производится в блоке <head></head>.
